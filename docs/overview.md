# Overview

Gigi IA = motor emocional-estético de Aurethica.
- **Core**: detección/calibración de tono (sinceridad, suavidad, firmeza, empatía, proximidad).
- **UX**: test breve + bloque de confirmación dinámico → vector tonal 0–1 por eje.
- **Salida**: respuestas y UI adaptadas al vector.

Carpetas:
- `dataset/` esquema y seed de calibración
- `design/` identidad, logo y spinners
- `app/` demo Next.js
- `api/` OpenAPI para integraciones
