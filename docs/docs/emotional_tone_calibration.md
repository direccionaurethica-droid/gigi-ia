# Calibración emocional

Ejes (0–1):
- sinceridad
- suavidad
- firmeza
- empatía
- proximidad

Cálculo: Likert 1–5 → (x-1)/4

Fuentes:
1. Bloques 1–4 del test
2. Bloque 5 dinámico (25 frases)
3. Texto libre (“¿Cómo quieres que te hable hoy?”)
